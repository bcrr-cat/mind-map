# 思维导图编辑器

一个基于HTML/CSS/JavaScript的在线思维导图编辑工具，支持节点颜色自定义、历史记录、导出PNG/PDF等功能。

## 功能特点

### 核心功能
- ✅ 创建和编辑思维导图
- ✅ 添加/删除节点
- ✅ 拖拽移动节点（子节点自动跟随）
- ✅ 双击编辑节点文本
- ✅ 右键菜单操作

### 颜色功能
- ✅ 同一级别节点自动使用相同颜色
- ✅ 支持手动更改节点颜色（15种渐变色可选）
- ✅ 自定义颜色不会影响同级别其他节点

### 视图控制
- ✅ 鼠标滚轮缩放（以鼠标位置为中心）
- ✅ 按住鼠标拖动平移画布
- ✅ 放大/缩小/重置按钮

### 数据管理
- ✅ 自动保存到浏览器本地存储
- ✅ 支持保存到本地文件（File System Access API）
- ✅ 历史记录列表（点击加载）
- ✅ 撤销功能（Ctrl+Z，最多50步）

### 导出功能
- ✅ 导出PNG图片
- ✅ 导出PDF文件

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Tab` / `Enter` | 添加子节点 |
| `Delete` | 删除节点 |
| `Ctrl+Z` | 撤销 |
| `双击` | 编辑文本 |
| `右键` | 打开菜单 |
| `鼠标滚轮` | 缩放视图 |

## 使用方法

### 直接使用
1. 双击 `index.html` 文件
2. 在浏览器中打开编辑器
3. 点击"+ 新建"创建思维导图

### 设置本地存储（推荐）
1. 点击工具栏"设置存储位置"按钮
2. 选择 `D:\mind map\history` 文件夹
3. 之后历史记录会自动保存到本地文件

## 技术栈

- HTML5 - 页面结构
- CSS3 - 样式设计
- JavaScript (ES6+) - 核心逻辑
- SVG - 绘制连接线
- Canvas API - PDF导出
- jsPDF - PDF生成库
